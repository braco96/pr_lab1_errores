README - Tema 1: Errores y Representación en Coma Flotante
=================================================================

Este paquete contiene los scripts y funciones MATLAB asociados al Tema 1 de Algorítmica Numérica.
Ningún código ha sido modificado; los archivos se incluyen como referencia para su análisis y desarrollo posterior.

Contenido:
-----------
1. README.txt             → Descripción y estructura del trabajo.
2. derivada_error.m       → Derivada numérica y estudio del error relativo.
3. lorentz_3D.m           → Generación del atractor de Lorentz y análisis de sensibilidad.
4. pi_arqimedes.m         → Aproximación numérica de π mediante polígonos inscritos/circunscritos.
5. log_precision.m        → Evaluación de log(1+x) y comparación entre precisión simple y doble.
